Требования к памяти 
--------------------
 <> Требования к памяти для Norton Commander.

 Для работы Norton Commander необходимо не менее 512K памяти для 
 работы всех утилит просмотра. Некоторые утилиты просмотра требуют 
 дополнительной памяти для работы в режиме быстрого просмотра. 

 <> Запуск Norton Commander с опциями памяти  Small и Tiny

 В некоторых ситуациях, связанных с нехваткой оперативной памяти,
 быстрый просмотр больших файлов при помощи утилит просмотра NC
 невозможен. С этой целью в новой версии предусмотрены две новые
 модели памяти, вызываемые из командной строки. В каждом из этих 
 режимов ограничивается число файлов, одновременно отображаемых 
 в каждой панели Norton Commander. Опции командной строки:

 Опции                   Описание
 -------------------     --------------------------------------------
 nc /s                   Запуск NC в режиме SMALL конфигурации
    памяти. Число файлов - 700.

 nc /t                   Запуск NC в режиме TINY конфигурации
    памяти. Число файлов - 700.
    
Поддержка сжатия файлов
-----------------------
 <> Сжатие скрытых файлов

 При сжатии скрытых файлов при помощи встроенной утилиты
 сжатия NC они потеряют атрибут "Скрытый". Это сделано для
 поддержания совместимости с форматом PKZIP. В настоящее
 время этот формат не поддерживает сжатие скрытых файлов.

 <> Использование пароля при сжатии файлов
 
 Встроенная утилита сжатия Norton Commander поддерживает
 совместимость с форматом PKZIP 1.x и PKZIP 2.x. При этом
 разрешается иметь один пароль для каждого файла-архива.

 <> Ограничения сжатия файлов
 
 Если в текущей панели открыт для просмотра файл-архив, то при
 работе в нем нельзя перейти на другой диск при помощи мыши.
 Для выполнения этой операции необходимо сперва закрыть 
 файл-архив, после чего выполнить эту операцию в обычном порядке.

 К сожалению, некоторые функциональные возможности Norton Commander
 нельзя реализовать в полном объеме для отдельных утилит сжатия.
 Особенно это относится к программам, которые не поддерживают
 распаковку файлов в каталог, отличный от текущего. Кроме того, 
 некоторые утилиты сжатия не поддерживают опцию сохранения полного
 имени и каталогов.

 Рекомендуем обратиться к заголовку файла Packer.set для получения 
 дополнительной информации о настройках утилит сжатия.

Поддержка утилит просмотра
--------------------------
 <> Встроенная программа просмотра
 
 Для запуска встроенной в NCMAIN.EXE утилиты просмотра 
 нажмите ALT+F3.

 <> Определение утилит просмотра
 
 Все утилиты просмотра, поддерживаемые Norton Commander версия 5.5,
 определены в файле NC.CFG. Пользователь может внести изменения в
 этот файл, добавив свои расширения файлов и связав их с определенными
 утилитами просмотра. Например, если текстовый процессор создает
 файлы с расширением .DCC, то для использования утилиты просмотра 
 текстовых файлов wpview.exe необходимо внести следующие изменения в
 соответствующий раздел: 
    
    ~wpview.exe
    $*.exe,p
    $*.com,p
    $*.dll,p
    $*.dcc,p     <----- Строка, добавленная для
    $*.*                обработки файлов *.dcc
Фильтры
-------
Параметры фильтра панелей, использовавшихся в  Norton Commander 4.0
хранились в файле NC.CFG. В новой версии программы эти фильтры и фильтры 
операций хранятся в двоичном файле NC.FIL. Если пользователь вносил 
изменения в определения фильтров в версии 4.0, то эту операцию надо 
повторить в новой версии, используя встроенные возможности настройки 
фильтров. Более подробная информация о настройке фильтров содержится 
в руководстве пользователя.

 совместного использования COM порта самой программой клонирования и 
 программой удаленной загрузки. Это приводит к появлению сообщения
 об ошибке. Для устранения этой проблемы используйте команду

Специальные символы в редакторе Norton Commander 
-------------------------------------------
Для ввода специальных символов иногда необходимо использовать комбинацию 
клавиш CTRL+Q с последующим вводом необходимой комбинации клавиш.
Например, для того чтобы ввести символ конца страницы надо  
нажать CTRL+Q, а затем ALT+12. 

Переменные среды Norton Commander 
---------------------------------------
Norton Commander использует три переменные среды для определения 
места расположения и хранения временных рабочих файлов: TEMP, TMP, NC.

Иностранные символы в именах файлов 
-----------------------------------------------

В сетях Novell, при использовании NDOS, существует возможность создания
различных имен файлов с использованием символов верхнего и нижнего
регистра из второй половины ASCII таблицы. Например команда:

MD КАТАЛОГ
MD каталог

создаст два новых каталога. По этой причине, Norton Commander не изменяет
регистр вводимых символов в именах файлов и каталогов, если они относятся
ко второй половине ASCII таблицы.

Совместимость с Windows 95 и OS/2
---------------------------------

При работе с NC в Windows '95 или OS/2 2.x и 3.x могут возникнуть трудности
при работе с дисковыми утилитами, в особенности с утилитой форматирования
и системной информацией.

Кроме этого, копирование фалов с длинными именами приведет к тому, что эти
длинные имена будут утеряны. Это верно как для Windows '95 так и для OS/2.

Утилита копирования дискет
--------------------------

Опция настройки "Пропускать плохие сектора" никогда не сохраняется в INI файле. 
По умолчанию она всегда выключена при запуске программы. Все остальные опции 
сохраняются.

Утилита воспроизведения звука
-----------------------------

В состав Norton Commander 5.5 входит утилита проигрывания файлов WAV. Т.к. эта
программа не использует дополнительного оборудования, кроме встроенного в
компьютер динамика, качество звука может быть не очень хорошим.

Утилита NC_EXIT
---------------

Эта полезная программа выгружает ядро Norton Commander без использования
резидентных программ. Она может быть использована для запуска дополнительных
программ. Более подробная информация содержится в файле NC_EXIT.DOC.

Novell NDS 
----------

Сетевые утилиты работают в Netware 3.x and 4.x. Утилита присоединения к серверу
не работает полностью на Netware 4.x серверах, использующих Novell Directory
Services(NDS).

Полезные советы по использованию Norton Commander в системе с 1MB памяти
------------------------------------------------------------------------

Система управления памятью, используемая в новой версии Norton Commander, 
сдвигает ядро программы (NCMAIN) в дополнительную или расширенную 
память для запуска внешних утилит. Если ОЗУ системы менее 1 Мбайта, то
сдвиг осуществляется на диск в специальный SWAP файл, расположенный в
каталоге временных файлов, описанном в одной из переменных среды:
TEMP, TMP, NC, NORTON.

При запуске Norton Commander с гибкого диска в системе с 1 Мбайт ОЗУ, 
для достижения максимальной производительности, используйте одну из этих 
переменных, указав в ней каталог на жестком диске.  В противном случае,
Norton Commander попытается создать этот файл на гибком диске.

Запуск Norton Commander с разделяемого сетевого диска
----------------------------------------------------

Norton Commander может быть запущен с разделяемого сетевого диска. Для этого:

1. Установка на сетевой диск
 - Войдите на сервер как Supervisor.
 - Создайте каталог NC5NET.
 - Установите Norton Commander в этот каталог со стандартных
   дистрибутивных  дискет.
 - Создайте пользователя с именем NC5NETUSER.
 - Убедитесь, что этот пользователь имеет только права чтения и 
   просмотра каталога NC5NET.

2. Установка NC 5.5 на компьютер пользователя.
 - Скопируйте только INI файлы из каталога NC5NET в каталог NC на
   компьютер пользователя.
 - Внесите изменения в файл autoexec.bat и установите следующие 
   переменные среды:
  SET NC=c:\NC5
  SET NORTON=c:\NC5
 - войдите в сеть под именем NC5NETUSER и убедитесь, что каталог NC5NET 
   включен в путь.
 - Наберите NC в приглашении DOS.

Использование меню команд
-------------------------

При создании меню команд код "Назначенная клавиша" может быть введен только
в латинском регистре клавиатуры. Русские символы блокированы для того, чтобы
избежать неоднозначности, когда одной клавише могут быть приписаны различные
команды.

Создание меню утилит
---------------------

В новой версии NC существует возможность добавить меню "Утилиты" к
главному меню Norton Commander. Для этого создайте файл TOOLS.MNU и
поместите его в один каталог с  файлом NC.INI.

Каждая строка в TOOLS.MNU соответствует команде меню в следующем формате:

 Имя_команды_меню;Выполняемая_команда

Например:
 
 File Date;fd.exe
 Norton Control Centre;ncc.exe
 Norton Disk Doctor;ndd.exe
 Norton Disk Editor;de.exe
 Text Search;ts.exe

Число пунктов в меню ограничено лишь наличием свободной памяти.

Использование гашения экрана при работе в Windows 3.1
-------------------------------------

При работе в сеансе DOS в MS-Windows, графические утилиты гашения
экрана в Norton Commander отключаются, даже если одна из них выбрана
в настройке программы. В этой ситуации NC использует утилиту гашения, 
устанавливаемую по умолчанию.
